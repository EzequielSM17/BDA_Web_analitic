# Lecciones aprendidas

## Qué mejorar

1.  Como el proyecto está pensado solo para los usuarios registrados, debería añadirse una función para los usuarios anónimos y para detectar robots.
2. Habría que detectar los _status_ de las peticiones para saber por qué falla una petición.
3. También habría que medir el tiempo que tarda en procesarse la petición, para saber cuáles son las más lentas
4.  Si hay microservicios, estaría bien registrar el proceso que se ejecuta y toda la ruta que sigue, para mejorar la trazabilidad entre microservicios.
5. y muchas cositas mas que mejorar
## Siguientes pasos

El siguiente paso sería realizar el punto 1 de las mejoras, ya que a nivel técnico y de negocio es lo que más interesa.
